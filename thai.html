My Biggest Project!

1. หนูสร้างเซิร์ฟเวอร์ Minecraft เพื่อเอาไว้เล่นกับเพื่อนๆ!!

Minecraft เป็นวีดีโอเกมส์ที่สร้างโดยบริษัท Mojang โดยเป็นเกมส์ผู้เล่นสามารถผจญภัย, สร้างสิ่งต่างๆ และเอาชีวิตรอดในโลกของเกมส์ที่มีมอนเตอร์อันตรายมากมาย ผู้เล่นสามารถเลือกที่จะเล่นคนเดียว หรือกับผู้เล่นคนอื่นบนเซิร์ฟเวอร์ก็ได้ โดยผู้เล่นสามารถที่จะสร้างโลกในแบบของตัวเอง โดย Minecraft นั้น ยังมีหลากหลายเวอร์ชั่นให้เลือกเล่นได้บนหลายแพลตฟอร์ม

Minecraft Server เป็นซอร์ฟแวร์ที่ทำให้ผู้เล่นคนอื่นๆสามารถเล่นด้วยกันได้ ซึ่งรูปแบบของเซิร์ฟเวอร์สามารถเปลี่ยนแปลงได้ และอาจมีลูกเล่นต่างๆ เพิ่มเข้ามา

สิ่งที่ได้ทำในโปรเจคต์นี้?

ทำการตั้งเซิร์ฟเวอร์บน Virtual Machine ของ GCP (Google Cloud Platform)

หนูได้เลือก GCP VM เป็นเซิร์ฟเวอร์ของเกมส์ โดยเซิร์ฟเวอร์ของ GCP นั้นเป็น Linux เซิร์ฟเวอร์ ส่วนตัวสเปคของเซอเวอร์นั้น หนูได้เลือกให้ตรงกับ กับ Requirements ของตัวเกมส์ และในขั้นตอนนี้หนูก็ได้มีโอกาสลองใช้ feature ต่าง ๆ ของ GCP เช่นการติดตั้ง Firewall ให้กับ instance ด้วย ซึ่งตัว Minecraft Server นั้นจำเป็นที่จะต้องเปิด Port 25565 ก่อน ถึงจะเข้าเล่นได้

เชื่อมต่อไปยัง Linux Server!

สำหรับการใช้ Linux ในช่วงแรก ๆ นั้นก็มีสิ่งที่ทำให้สับสนอยู่หลายอย่าง เพราะมันต่างจาก Windows OS ที่มี Interface และ mouse กับ Keyboard ให้กด แต่พอได้ลองศึกษาวิธีการใช้ขั้นต้น เช่น คำสั่งพื้นฐาน การเปลี่ยน Directory การแก้ไขไฟล์ด้วย Vim การ copy/paste และอีกหลายๆอย่าง ซึ่งข้อมูลพวกนี้ก็เพียงพอสำหรับในการเปิด Minecraft เซิร์ฟเวอร์ได้ค่ะ

ทำการติดตั้ง Minecraft Server

หลังจากที่ได้ Linux Server มาแล้วก็ทำการดาวน์โหลด Minecraft Server มาจาก Mojang และก็ทำการตั้งค่าของเซิฟเวอร์นิดหน่อยก่อนที่จะ run ตัวเซิฟเวอร์ เช่นต้องการแบ่ง Ram ให้ตัวเกมส์เซิฟเวอร์เท่าไหร่ ซึ่งต้องสอดคล้องกับสเปคของ Server ของเรานะคะ ไม่อย่างนั้นเซิฟเวอร์จะ Freeze ได้ค่ะ หลังจากเซิฟเวอร์รันได้แล้ว ก็ส่ง Ip ของเซิฟเวอร์ไปให้เพื่อน ๆ เข้ามาเล่นด้วยกันได้เลยค่ะ !!

บำรุงรักษาเซิฟเวอร์

หลังจากที่เพื่อน ๆ เข้ามาเล่นในเซิฟเวอร์ได้กันแล้ว แต่ระหว่างเล่นไปนั้น ก้เกิดปัญหาต่าง ๆ อยู่เรื่อย ๆ เช่นเซิฟเวอร์ค้าง เพราะมีผู้เล่นเข้ามาเล่นพร้อมกันเยอะเกินไป เลยต้องทำการ Restart เซิฟเวอร์ และบางครั้งก็แก้ปัญหาด้วยการปรับเพิ่ม RAM ของ GCP VM ค่ะ

ถ้าต้องการมาเล่นด้วยกัน สามารถเข้ามาได้ที่ IP นี้เลยค่ะ (แต่ต้องมีตัวเกมส์ ก่อนนะคะ)

2. เว็ปไซต์ sirikarn.com

เว็ปนี้หนูสร้างขึ้นมาเพื่อนำเสนอความสามารถและความสนใจที่หนูมีต่อสาขาคอมพิวเตอร์ค่ะ ซึ่งการทำเว็ปไซต์นี้ขึ้นมา ทำให้หนูได้เรียนรู้ทั้งการทำ Frontend และ Backend ค่ะ (และทำให้หนูได้รู้ถึงความยุ่งยากของ Backend เลยค่ะ ☹️)

ได้ทำอะไรในโปรเจคต์นี้ ?

ทำการตั้งเซิฟเวอร์ บน Virtual Machine ของ Google Cloud Platform

การตั้งเซิฟเวอร์ของเซิฟนี้ก็คล้ายกับโปรเจคต์ก่อนหน้านี้ โดยใช้ GCP เป็น Cloud Platform

ตั้งค่า Server บน Linux

ในส่วนของการ Host เว็ปเซต์นั้นหนูเลือกที่จะใช้ Apache เป็น Web Server ค่ะ เพราะว่าจากการลองศึกษาดูแล้ว Apache มีการใช้งานที่ง่าย และมีฟังก์ชั่นที่ครบตามที่ต้องการค่ะ

เว็ปไซต์กำลัง Online อยู่! ...แต่ยังไม่เสร็จค่ะ

ตอนนี้ถึงแม้ว่าเว็ปไซต์จะสามารถเข้าไปดูได้แล้ว แต่หน้าเว็ปยังคงเป็นแค่หน้า Default ของ Apache อยู่ค่ะ ซึ่งสามารถเข้าไปดูได้ด้วย IP ของ เซอเวอร์

ทำการซื้อ Domain name

เนื่องจากหนูอยากให้เว็ปไซต์นี้มีชื่อเหมือนกับหนู ไม่ใช่เป็นแค่ตัวเลข IP หนูเลยทำการซื้อ Domain Name ชื่อ "sirikarn.com" มาจาก Google Domain เป็นเวลา 1 ปีค่ะ

ตั้งค่า DNS server

หลังจากที่ได้ Domain Name มา หนูจำเป็นที่จะต้องไปตั้งค่าให้ DNS ชี้มายัง GCP ค่ะ ซึ่งขั้นตอนนี้หนูคิดว่ามันยากและสับสนมาก ต้องใช้เวลาอยู่นานจนกว่าจะเจอวิธีการที่ถูกต้อง ตามคำแนะนำบนอินเทอร์เน็ตค่ะ

ตั้งค่า ใบรับรอง HTTPS

หลังจากที่ได้มีการเข้าไปดูเว็ปไซต์บนมือถือ หนูพบว่ามันเกิด Error ว่าเว็ปของหนูนั้นไม่ปลอดภัย จึงได้ทำการศึกษาหาคำตอบจากอินเทอร์เน็ตดู จึงพบว่ามันเกิดจากเซิฟเวอร์ของหนูยังไม่มีใบรับรอง HTTPS ต่อจากนั้นก็ทำการแก้ไขปัญหานี้ไปค่ะ ด้วยการใช้ Certbot ซึ่งขั้นตอนนี้นั้นมีความง่ายกว่าที่คิดมากค่ะ เพียงแค่คำสั่งเดียวที่ run ตัว Certbot ก็จัดทำใบรับรองจาก let's Encrypt ให้เรียบร้อย

เริ่มการเขียนเว็ปไซต์ !!

สำหรับการเขียนเว็ปไซต์นี้ขึ้นมานั้น ตอนแรกหนูพยายามที่จะเขียนโดยเริ่มจากหน้าเปล่า ๆ แต่พอลองทำดูแล้วหน้าตามันดูไม่ทันสมัยมาก ๆ เลยค่ะ 😀 หนูเลยทำการนำ Template จากอินเทอร์เน็ตมาแก้ไขดู ซึ่งทำให้หนูได้เรียนรู้เกี่ยวกับการใช้ ฺBoostrap ซึ่งทำให้การจัด UI นั้นง่ายมาก ๆ เพียงแค่ตั้งชื่อ Class ให้กับ Element ก็จะได้ UI สวย ๆ แล้วค่ะ

อยากเรียนรู้เรื่อง AI /Machine Learning

ในตอนนี้ที่ AI เป็นสิ่งที่ Programmer ทุกคนควรรู้ หนูจึงอยากจะเรียนเกี่ยวกับ machine learning, natural language processing และ computer vision ค่ะ หนูอยากที่จะเรียนรู้กับการนำเอา AI ไปประยุกต์กับเทคโนโลยีต่าง ๆ รอบตัวเพื่อพัฒนาให้ดีขึ้นค่ะ และหนูเชื่อว่าที่ มหาวิทยาลัยเกษตรศาสตร์นี้ หนูจะได้รับการศึกษาหรือได้ใช้ทรัพยากรของทางคณะ และให้เกิดความรู้ความเข้าใจด้าน AI เป็นอย่างมากค่ะ

อยากรู้การทำงานของ Internet

นอกจากเรื่อง AI แล้ว หนูก็มีความสนใจในด้าน Internet ค่ะ และอยากเรียนรู้เพิ่มทั้งการสร้าง Website และโครงสร้างของระบบเครือข่าย หนูเชื่อว่าในตอนนี้ Internet จะเป็นรากฐานของเทคโนโลยี ทำให้ความเข้าใจในส่วนนี้จะทำให้เราสามารถสร้างสิ่งใหม่ ๆ ได้อย่างมีคุณภาพค่ะ

ความสนใจในการพัฒนาวีดีโอเกมส์

หนูอยากได้โอกาศลองศึกษา พัฒนาและออกแบบวิดีโอเกมส์ดูค่ะ หนูอยากเรียนรู้วิธีการใช้ Game Engine ต่าง ๆ การเขียนโปรแกรมเพื่อพัฒนาเกมส์ และหลักการออกแบบเกมส์ค่ะ หนูเชื่อว่าถ้าได้เรียนที่มหาวิทยาลัยเกษตรศาสตร์แห่งนี้สมารถที่จะทำให้หนูพัฒนาความสามารถในด้านนี้ได้ค่ะ
